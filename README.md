# Suno 基本操作指南

一份給初學者的 Suno AI 音樂生成互動式教學，採莫蘭迪暖灰配色。

## 內容章節

1. 開始之前
2. 基本流程（三個欄位的分工）
3. Style 欄位的黃金公式
4. 常用樂器分類速查
5. 10 種主流音樂風格
6. 歌詞結構（七種段落）
7. `[ ]` 與 `( )` 的差異與用法
8. 互動式歌詞結構模擬器（拖曳組合）
9. 常見錯誤示範
10. 一頁速查表

## 線上瀏覽

如果有開啟 GitHub Pages：

👉 https://techtrekleo.github.io/suno-guide/

## 本機執行

直接用瀏覽器打開 `index.html` 即可，無需安裝任何依賴。

## 檔案結構

```
.
├── index.html              # 主頁面（與「Suno 基本操作指南.html」相同）
├── Suno 基本操作指南.html   # 中文檔名版本
├── styles.css              # 莫蘭迪配色樣式
└── app.js                  # 章節導航 / 拖曳模擬器 / 複製按鈕
```

## 設計

- **配色**：莫蘭迪暖灰系（#EDE8E0 / #5A5248 / #A89888）
- **字體**：Noto Sans TC + JetBrains Mono
- **互動**：可複製 prompt、對照切換、拖曳歌詞結構模擬器
